i2c-usb
---------------------------------------

The firmware code itself is distributed under the GPL.

The default configuration is for a atmega8. The Makefile
allows to compile the device for the Atmega8 cpu. This includes
the possibility to use the atmega8 for debugging.


This project supports usb implementations:

To Create hex file, type
           make all

To compile and upload the file atmega, type
           make avrcompile

To clean, type
           make clean
